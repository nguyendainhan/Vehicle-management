# Parking System (Vue + Express + MySQL)

## 1. Clone project
git clone https://github.com/nguyendainhan/Vehicle-management.git

## 2. Install Backend

cd Backend

npm install

## 3. Import database
mysql -u root -p parking_system < parking_system.sql

## 4. Create file .env
copy .env.example .env

## 5. Run backend
node server.js

## 6. Run frontend
cd Frontend

npm install

npm run dev
